HOW TO INSTALL THE GAME:
--------------------------------------------------------------------------------
1. Install required libraries on your computer like this depending on your os:
  1.1 Windows: 
  1.2 Ubuntu-like OS: Run these commands in shell
   >sudo apt-get update
   >sudo apt-get install libsdl2-dev libsdl2-image-dev make
2. Run Makefile in shell like this depending on your os:
  1.1 Windows:
  1.2 Linux: Right click in root folder, click on "Open terminal/shell here"
   >make
3. The game is installed. To run it do like this depending on your os:
  3.1 Windows: Search Roguemon.exe in bin folder and run it
  3.2 Linux: Right click on bin folder, click on "Open terminal/shell here"
   >./roguemon
